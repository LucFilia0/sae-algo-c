# SAE algo & C

Cette SAE (Situation d'Apprentissage et d'Évaluation) avait pour but de créer une application console en language C, afin de permettre la gestion d'un aéroport.
L'application devait afficher les vols, les trier selon leur heure, les filtrer selon leur compagnie.
Les allgos utilisés étaient vus en cours d'algorithmie, comme la dichotomie, le tri fusion, etc...
